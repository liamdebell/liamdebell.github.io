Decision time.

When I first thought of building my own computer, I never envisioned a flawless machine that functioned immediately without fault. Deciding and matching up the components was arguably the most difficult part of the process, making sure that the CPU would function in harmony with RAM and Secondary Storage. Making the decision was not easy, as analysing each of the components specifications was necessary, to make the best optimised computer available. Now 4 years later, the components, however dated still function perfectly without issues and I am still able to play games and other basic functions, like programming this website!

Factors

Once I recieved all of my components, I was finally able to begin construction. This suprisingly went without issue and after installing the base components such as: RAM, CPU, PSU, MOTHERBOARD and GRAPHICS, I was able to power on the Machine for the first time.
