ImGui module

ImGui is a graphical user interface module developed for c++, specifically compiled through the .dll file extension, or engine file. This is injected into a program and runs as an overlay, similar to how a game would interpret messaging apps such as discord. It allows sections of code known as voids or classes, to run on the event of a button press or keybind. Specifically this menu type is very unique, with its hundreds of features, including a test window, that allows you to customise your colour scheme and features and further export them to code format, so that it can be permanently installed in your configuration file.
