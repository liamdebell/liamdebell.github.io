NAS Home Network independent

Purchasing a 5ghz wifi antenna allowed me to connect the NAS remotely to any device, this allowed much more flexibility amoung what I could do with it. The system I used can be seen here, alongside some of the key components:

Network Attatched Storage Home Build

Developing a private network for use in my home was very interesting, as it meant I could get an in depth look inside the basic mechanics of networking, and how each component in the system contributed to the load. I used an open source operating system called “Free Nas”, this was great for my intent, as it was not only free, but easy to modify and update. To begin with I purchased an old system running windows 7 with poor specs, updating the RAM to 8gb and installing 3 hard drives for storage use and backups. Alongside this I burnt a copy of “Free Nas” iso to a disc so that I could further install the operating system and mount each drive appropriately.

A local host would be available after proper installation and from there you could logon and view each of your drives and their stats. Using SSH I opened the necessary ports to allow a TCP connection from a different network and logon with a user name and password. The servers’ main purpose was to host movies so that my Father and I could watch them with out having to get the disc. Connecting to the server on my phone, would allow access to the partitions made allowing access to the movies installed. From there you could stream almost any movie you wanted without lag and in HD.

Factors

  • FreeNas optimisation
  • User authentication
  • High capacity
  • Fast read and write
